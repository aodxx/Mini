# 🎨 Interactive UI Library

คอลเลกชัน Interactive UI Components ที่สร้างด้วย AI สำหรับเรียนรู้และใช้งานจริง

## 📦 ประเภทของ Interactive

### 1. 🗂️ Navigation-Based
- [Tabs Example](01-navigation/tabs-example.html) - สลับดูเนื้อหาด้วยแท็บ
- [Accordion Example](01-navigation/accordion-example.html) - ขยาย-หุบเนื้อหา
- [Carousel Example](01-navigation/carousel-example.html) - เลื่อนดูรูปภาพ

### 2. 📈 Data Visualization
- [Line Chart](02-data-visualization/chartjs-line.html) - กราฟเส้นแบบ interactive
- [Bar Chart](02-data-visualization/chartjs-bar.html) - กราฟแท่ง hover ได้
- [Mindmap](02-data-visualization/d3-mindmap.html) - แผนผังความคิด

### 3. 🕸️ Relationship
- [Flowchart](03-relationship/mermaid-flowchart.html) - แผนผังกระบวนการ
- [Org Chart](03-relationship/org-chart.html) - โครงสร้างองค์กร
- [Network Graph](03-relationship/network-graph.html) - กราฟเครือข่าย

### 4. 🎮 Simulation
- [Quiz App](04-simulation/quiz-app.html) - แบบทดสอบมีคะแนน
- [Flashcards](04-simulation/flashcards.html) - บัตรคำพลิกได้
- [Step Tutorial](04-simulation/step-tutorial.html) - สอนแบบทีละขั้น

### 5. 🎛️ Calculator
- [Loan Calculator](05-calculator/loan-calculator.html) - คำนวณเงินกู้
- [BMI Calculator](05-calculator/bmi-calculator.html) - คำนวณดัชนีมวลกาย
- [Unit Converter](05-calculator/unit-converter.html) - แปลงหน่วย

### 6. 🎨 Creative
- [Color Picker](06-creative/color-picker.html) - เลือกคู่สี
- [Font Preview](06-creative/font-preview.html) - พรีวิวฟอนต์
- [Layout Designer](06-creative/layout-designer.html) - ออกแบบ layout

## 🛠️ เทคโนโลยีที่ใช้

- **HTML5** - โครงสร้าง
- **Tailwind CSS** - Styling
- **JavaScript (Vanilla)** - Logic
- **Chart.js** - กราฟ
- **Mermaid.js** - Flowchart
- **D3.js** - Visualization ซับซ้อน

## 📖 วิธีใช้งาน

1. คลิกที่ลิงก์ของ Interactive ที่ต้องการ
2. ถ้าเปิดใน GitHub ไม่ได้ ให้คลิก **"Raw"** แล้ว save เป็นไฟล์ `.html`
3. เปิดไฟล์ในเบราว์เซอร์

หรือใช้ **GitHub Pages** (ดูวิธีด้านล่าง) เพื่อเปิดเป็นเว็บไซต์จริง
